#프로젝트 명 : Offinity

## 팀원
1. 내 정보 - 이현진
2. 연차/근태 관리 - 홍중택
3. 직원 명부 - 백승하
4. 사원자유게시판 - 서지민 (팀장)
5. 건의게시판 - 김현조
6. 공지게시판 및 로그인/회원가입 - 박은진
7. 캘린더 -이희정
